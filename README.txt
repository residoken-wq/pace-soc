SOC Agent Pack
=================

This pack contains:
- README.txt
- bootstrap-soc-agent.sh

Usage:
1. Upload both files to any Linux host.
2. Run: sudo bash bootstrap-soc-agent.sh
3. Agent will automatically connect to SOC Center (192.168.1.206).

Override SOC IP:
SOC_CENTER_IP=115.79.5.5 sudo bash bootstrap-soc-agent.sh
